# Loterias Brasil 

## Tecnologias Utilizadas
- HTML5
- CSS3 (Variaveis CSS, Gradientes, Animacoes)
- JavaScript (Contador animado, Smooth scroll)
- Squeleton CSS Framework
- WOW.js (Animacoes de scroll)
- Google Fonts (Montserrat)

## Funcionalidades Principais
- **Hero Section**: Apresentacao com bolas de loteria animadas
- **Estatisticas**: Cards com numeros (6 modalidades, R$300Mi+ acumulados, 1500 sorteios/ano)
- **Cards de Loterias**: Informacoes detalhadas de cada modalidade
- **Como Jogar**: Passo a passo em 3 etapas
- **Ultimos Resultados**: Cards com os numeros sorteados de cada loteria
- **Menu Mobile**: Modal para navegacao em dispositivos moveis

## Loterias Apresentadas
1. **Mega-Sena**: 6 de 60 numeros, sorteios Qua/Sab
2. **Lotofacil**: 15 de 25 numeros, sorteios Seg a Sab
3. **Quina**: 5 de 80 numeros, sorteios diarios
4. **Lotomania**: 50 de 100 numeros, sorteios Ter/Qui/Sab
5. **Dupla Sena**: 6 de 50 numeros (2 sorteios), Ter/Qui/Sab
6. **Timemania**: 10 de 80 numeros + time, Ter/Qui/Sab

## Design
- Paleta de cores: Dourado (#f39c12), Preto (#1a252f), cores especificas por loteria
- Bolas 3D com efeito de brilho
- Cards com bordas superiores coloridas
- Layout responsivo
